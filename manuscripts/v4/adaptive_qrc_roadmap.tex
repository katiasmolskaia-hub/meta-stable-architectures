\section*{Adaptive RC Roadmap}

\begin{itemize}
    \item What we add: a state-aware instructor model for group synchronization.
    \item Where it enters: as a closed-loop coordinator that reads dispersion, lag, suppression, and contamination.
    \item Why it matters: the coordinator adapts rhythm and focus instead of forcing uniform behavior.
    \item What we keep separate: memory stays in $K$; suppression stays in $S$; coordination stays in RC.
\end{itemize}

\paragraph{1. Core role.}
The adaptive coordinator is not a fixed metronome. It is a soft controller that changes its own gain and focus according to the group response. If dispersion and lag rise, the coordinator strengthens guidance. If the group stabilizes, guidance is relaxed.

\paragraph{2. Variables.}
\begin{itemize}
    \item $S$ --- suppression, isolation, or temporary unreadiness;
    \item $K$ --- experience, bounded memory, consolidation;
    \item $\mathcal{D}$ --- phase dispersion;
    \item $L$ --- lag or delayed adaptation;
    \item $A$ --- stable anchors;
    \item $C$ --- contagion or propagation of instability.
\end{itemize}

\paragraph{3. What each variable does.}
\begin{itemize}
    \item $S$ tells the coordinator which agents are not yet ready for strong coupling.
    \item $K$ records accumulated experience and saturates instead of growing without bound.
    \item $\mathcal{D}$ measures how far the group has drifted from common rhythm.
    \item $L$ indicates which subgroup is slow to adapt.
    \item $A$ identifies stable agents that can serve as anchors.
    \item $C$ flags the risk that a local failure spreads to neighbors.
\end{itemize}

\paragraph{4. Minimal closed-loop update.}
\begin{align}
\phi_{\mathrm{gain}}(t) &= \mathrm{clamp}\!\left(\phi_0\bigl[1+a_1\mathcal{D}(t)+a_2\overline{L}(t)+a_3 C(t)\bigr]\right), \\
\Phi_{t+1} &= \Phi_t + \Delta t\left[-\kappa(\mathcal{D},L)\Phi_t + \mathrm{Arg}\bigl(\langle e^{i\theta}\rangle\bigr)\right].
\end{align}
This is the minimal form of the instructor model: state is observed, gain is adapted, and the global rhythm is shifted rather than enforced.

\paragraph{5. Focus and gating.}
Stable agents should receive more weight, while unstable neighbors should contribute less.
\begin{align}
w_i &\propto (1-S_i), \\
\varepsilon_{ij} &= \varepsilon(1-S_i)(1-S_j).
\end{align}
This implements two rules:
\begin{itemize}
    \item look first at agents that remain stable;
    \item reduce contagion from agents that are still collapsing.
\end{itemize}

\paragraph{6. What to test.}
\begin{itemize}
    \item baseline RC vs adaptive RC;
    \item recovery time under different lag profiles;
    \item whether the adaptive version reduces critical-noise sensitivity;
    \item whether stable anchors improve recovery in mixed-experience groups.
\end{itemize}

\paragraph{7. Writing rule.}
Keep the roles separated:
\begin{itemize}
    \item $S$ is not memory;
    \item $K$ is not control;
    \item RC is not hard forcing;
    \item the adaptive layer only reads state and changes its gain.
\end{itemize}
