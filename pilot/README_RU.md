# Pilot (RU): узкий пилот метаслоя

## Что это

Минимальный эксперимент "baseline vs metaslayer_v0" на одинаковом наборе задач.

- `baseline_agent.py` - агент без метаслоя.
- `metaslayer_v0.py` - агент с простым кризис-режимом.
- `run_pilot.py` - запуск и сравнение метрик.
- `results/*.csv` - результаты по шагам.

## Метрики

1. `success_rate` - доля успешно решенных задач.
2. `failure_rate` - доля срывов/ошибок.
3. `avg_time` - среднее время на задачу.

## Запуск

```bash
python pilot/run_pilot.py
```

## Что ожидаем

Метаслой обычно:
- снижает `failure_rate` на стресс-участках,
- иногда слегка увеличивает `avg_time` (плата за устойчивость).

## Как крутить

В `metaslayer_v0.py`:
- `trigger` - когда включается кризис-режим,
- `damp_strength` - насколько сильно "гасим" нагрузку,
- `tau` - длина памяти кризиса.
